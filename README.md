# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in here: https://gametet.onrender.com

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Google ai API key
3. Run the app:
   `npm run dev`
